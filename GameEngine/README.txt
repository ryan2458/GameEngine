Directions for Game Demo (Asteroids)
------------------------------------
Up Arrow - Forward movement
Down Arrow - Backward movement
Left Arrow - CounterClockwise rotation
Right Arrow - Clockwise rotation


Required Libaries and Includes: https://drive.google.com/file/d/1TT4KrDzJF3TSo4Fc3s2wXqAfXYFVhELW/view?usp=sharing
Don't forget to add libraries and includes in your C++/Linker settings in Visual Studio!

To get sound working
--------------------
Libs_Includes\irrKlang\bin\winx64-visualStudio\ikpMP3.dll
Libs_Includes\irrKlang\bin\winx64-visualStudio\irrKlang.dll
Add both of these to the Debug folder that contains the project executable
